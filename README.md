# Install

```
git clone git@github.com:Olyno/rattrapage-osint-python.git osint
cd osint
git submodule update --init --recursive --remote
pip3 install -r requirements.txt
```

One line:

```
git clone git@github.com:Olyno/rattrapage-osint-python.git osint && cd osint && git submodule update --init --recursive --remote && pip3 install -r requirements.txt
```

# Usage

```
python3 ./main.py <domain>
```
